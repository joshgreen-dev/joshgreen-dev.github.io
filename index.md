I build tools that make 3D files easier to work with. My main project is GeometryViewer, a browser-based 3D viewer. Since Microsoft is removing the [deprecated Windows 3D Viewer](https://geometryviewer.com/windows-3d-viewer-alternative), I'm building a replacement that handles STL, OBJ, GLB, and 3MF without installing anything. I also work on ModelDirectory.org, a search engine where I recently published a [data analysis of 246,000 3D models](https://modeldirectory.org/blog/we-analyzed-250000-3d-models.html) across Thingiverse, Printables, MyMiniFactory, and Sketchfab.
