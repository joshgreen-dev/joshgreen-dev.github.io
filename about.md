Web developer. Bambu P1S owner. I build browser-based tools for 3D files.

Currently working on two things:

- **[GeometryViewer](https://geometryviewer.com)** - free 3D file viewer that runs in the browser. STL, OBJ, GLB, GLTF, 3MF. AR preview on mobile. No install. I built the rendering engine and tested it against 250,000 files from Thingiverse and Printables.

- **[ModelDirectory.org](https://modeldirectory.org)** - search engine that aggregates 246,000 3D models across four platforms. I built the crawler, search pipeline, and AI classification system.

Before this I did Shopify theme dev and frontend JS. The 3D viewer started as a side project for an e-commerce client and grew from there.

I write here about 3D printing, local LLMs, WebGL, and whatever else I'm tinkering with.
